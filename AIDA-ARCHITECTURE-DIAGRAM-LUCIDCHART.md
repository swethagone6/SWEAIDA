## Data Flow Description

```
1. User enters question in Streamlit
2. Streamlit sends HTTPS request to API Gateway
3. API Gateway authenticates, authorizes, validates
4. API Gateway invokes Lambda function
5. Lambda initializes LangChain agent
6. Agent calls Claude via Bedrock
7. Claude generates SQL with retry logic
8. Lambda executes SQL on Athena
9. Athena queries S3 data
10. Results returned to Lambda
11. Claude formats answer
12. Lambda returns response to API Gateway
13. API Gateway returns to Streamlit
14. Streamlit displays answer to user
15. Memory stores conversation context
```
